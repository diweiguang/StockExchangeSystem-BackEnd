## 模拟股票交易系统后端

采用Flask框架，前后端分离开发股票交易系统

[前端部分](https://github.com/CrazyQii/StockExchangeSystem-Front)

### 创建Python虚拟环境

- 下载Python虚拟环境

````bash
pip install pipenv
```
````

- 进入目录项，为当前项目创建虚拟环境，并下载需要的依赖包

```bash
pipenv install
```

- 加载环境变量

```bash
pipenv shell
```

### 运行程序

```bash
python run.py
```
